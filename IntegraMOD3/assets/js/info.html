Information related to javascript file. Both the Stargate Portal and the Kiss Engine use a common folder (root/js) to store all javascripts... A copy of the default phpBB core javascript files are stored in this folder... phpBB scripts are updated for each new phpBB release, all other files are updated on release of new versions... Folder structure: root/js (miscellaneous file including a copy of the default javascript files used by the phpBB core)... root/js/drag_n_drop (scripts to support drag and drop for user block layouts)... root/js/jquery (most recent jQuery library files)...
